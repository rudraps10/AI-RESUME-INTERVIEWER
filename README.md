# 🎯 AI Resume Interviewer

AI-powered mock interview platform that generates personalized interview questions from a candidate's resume and provides detailed performance feedback.

## 🚀 Features

- Upload and analyze resumes
- AI-generated interview questions
- Multiple interview modes (HR, Technical, General)
- Interactive mock interview experience
- AI-based answer evaluation
- Performance scoring and feedback dashboard

## 🛠️ Tech Stack

**Frontend**
- React
- TypeScript
- Vite
- Tailwind CSS

**Backend**
- Supabase
- Supabase Edge Functions

## 📂 Project Workflow

1. Upload Resume
2. Select Interview Type
3. Generate AI-Powered Questions
4. Answer Questions
5. Receive Evaluation & Feedback

## ⚙️ Installation

```bash
git clone https://github.com/rudraps10/AI-RESUME-INTERVIEWER.git
cd AI-RESUME-INTERVIEWER
npm install
npm run dev
```
